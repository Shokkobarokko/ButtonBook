## 🚀 Как использовать

Каждая папка содержит `.swift` файл с кодом кнопки и `README.md` с объяснением.  
Открой нужный файл и запусти в **Xcode Preview** или на симуляторе.

## 📦 Технологии

- Swift 5.9+
- UIKit + Auto Layout (код)
- SwiftUI + ButtonStyle + Animations
- iOS 16+ (кроме Metal Shader — iOS 17+)
