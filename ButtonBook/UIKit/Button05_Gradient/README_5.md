# Button 05 — Gradient

**Уровень сложности:** ⭐⭐☆☆☆
**Фреймворк:** UIKit
**iOS:** 13+

## Что делает
Кнопка с градиентным фоном (синий → фиолетовый). При нажатии слегка сжимается и теряет прозрачность, при отпускании «пружинит» обратно.

## Ключевые техники
- `CAGradientLayer` — слой градиента
- `clipsToBounds` — обрезка содержимого по границам view
- `startPoint` / `endPoint` — направление градиента
- `CGAffineTransform` — масштабирование при нажатии
- Пружинная анимация `usingSpringWithDamping`

## Как запустить
Запустить проект, в главном меню выбрать **"Gradient"**.

## Что изучишь
- Добавление слоёв (`CAGradientLayer`) к `UIView`
- Правильный момент для установки размеров слоя (`viewDidLayoutSubviews`)
- Разница между `viewDidLoad` и `viewDidAppear`
- Трансформации вьюх (`CGAffineTransform`)
