# Button 02 — Filled

**Уровень сложности:** ⭐☆☆☆☆  
**Фреймворк:** UIKit  
**iOS:** 16+

## Что делает
Кнопка с цветной заливкой, скруглёнными углами и внутренними отступами. При нажатии плавно затемняется.

## Ключевые техники
- `UIButton(type: .custom)` – полный контроль над оформлением
- `backgroundColor` и `layer.cornerRadius`
- `contentEdgeInsets` для отступов
- Отслеживание состояний `.touchDown`, `.touchUpInside` для анимации нажатия

## Как запустить
Запустить проект, в главном меню выбрать **"Filled"**.

## Что изучишь
- Настройка фона и скругления кнопки
- Управление отступами содержимого
- Простейшая анимация подсветки через изменение `alpha`
