# Button 01 — Plain

**Уровень сложности:** ⭐☆☆☆☆  
**Фреймворк:** UIKit

## Что делает
Простая кнопка UIButton с системным стилем.

## Ключевые техники
- UIButton(type: .system)
- addTarget / @objc action
- Auto Layout (центрирование)

## Как запустить
Запустить проект, в главном меню выбрать "Plain".

## Что изучишь
Создание кнопки программно, подключение действия, базовые констрейнты.
